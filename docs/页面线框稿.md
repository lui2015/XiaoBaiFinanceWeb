> 断点：手机 ≤ 768px / 平板 769–1024px / PC > 1024px。

### 3.3 元素清单与交互

| 元素 | 状态 | 行为 |
| --- | --- | --- |
| ♡收藏 | 未登录 | 弹登录引导弹窗，登录成功后自动完成收藏 |
| ♡收藏 | 已登录·未收藏 | 点击 → 调用接口 → 变实心 + Toast「已收藏」 |
| ♡收藏 | 已登录·已收藏 | 再次点击 → 取消，二次确认（移动端长按） |
| ♥点赞 | 同上 | 同收藏逻辑 |
| 反馈 | 未登录 | 弹登录引导 |
| 反馈 | 已登录 | 弹半屏：选择类型（有用/没用/报错），报错支持 ≤200 字描述 |
| 目录 | 自动生成 | 基于正文 h2/h3，PC 右侧固定，移动端折叠到顶部「目录 ▾」 |
| 阅读量 | 曝光后 | 进入页面 3s 后异步上报；同会话同文 30min 内仅计 1 次 |
| 上一/下一篇 | 同分类内 | 无相邻则按钮置灰 |
| 分享（移动） | ⋯按钮 | 复制链接 / 微信 / 微博（可选） |

### 3.4 安全渲染

- 正文区域 `<article class="post">` 内渲染服务端净化后的 HTML。
- 前端使用 DOMPurify 再次净化（双保险）。
- 图片懒加载，外链全部转存到站内 CDN。

## 5. 登录引导弹窗（互动触发）

> 用户未登录时点击收藏/点赞/反馈/我的收藏等需要登录的操作时弹出。**不是独立路由**，是浮层。

### 5.1 线框

```
┌──────────────────────────────────────────┐
│             ╳ 关闭                       │
│                                          │
│         登录后即可收藏文章 📚            │
│   登录后还可以同步浏览历史、提交反馈     │
│                                          │
│   📱 手机号 [____________________]       │
│   🔑 验证码 [______]  [获取验证码 60s]   │
│                                          │
│   [    立即登录 / 注册    ]              │
│                                          │
│   ─────────  其他方式  ─────────         │
│                                          │
│   邮箱密码登录 ›    （微信登录 P1）      │
│                                          │
│   登录即同意 《用户协议》《隐私政策》    │
│   ☐ 我已阅读并同意                       │
└──────────────────────────────────────────┘
```

### 5.2 交互规则

| 场景 | 行为 |
| --- | --- |
| 触发 | 未登录用户点击 ♡收藏 / ♥点赞 / 反馈 / 「我的收藏」入口等 |
| 关闭 | 右上角 ╳ / 遮罩点击 / Esc，关闭后停留在原页面 |
| 登录成功 | 关闭弹窗 + 自动续作原操作（如收藏当前文章） + Toast 提示 |
| 协议 | 必须勾选才可点登录按钮，否则按钮置灰 |
| 验证码 | 60s 倒计时；失败提示「验证码错误，请重试 (剩余 N 次)」 |
| 风控 | 一定阈值后弹图形验证码（极验滑块） |
| 移动端 | 半屏抽屉从底部弹出，覆盖率 80%，可下拉关闭 |

# 通用规范

## 21. 状态规范

| 类型 | 规范 |
| --- | --- |
| Loading | 列表页骨架屏；按钮内 spinner；超过 5s 显示「正在加载…」+ 可取消 |
| 空状态 | 插画 + 一句引导 + 主操作按钮 |
| 错误页 | 404：返回首页；500：建议刷新 + 联系客服；网络异常：重试按钮 |
| Toast | 成功（绿√）/ 失败（红✕）/ 提示（蓝i），3s 自动消失 |
| 危险操作 | 二次确认弹窗（删除、封禁、注销）；输入「确认」字样可选 |
| 表单校验 | 失焦校验 + 提交校验；错误信息红字显示在字段下方 |

## 22. 移动端通用细节

- 点击区域 ≥ 44×44px；按钮高度 ≥ 44px。
- 顶部固定导航高度 48px；底部安全区适配 iOS 刘海屏。
- 关键页面支持「下拉刷新」：首页、分类列表、我的收藏、浏览历史。
- 网络异常时全局 Toast 提示，已加载内容继续可用（离线缓存）。
